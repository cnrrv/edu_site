Понятие адресации памяти компьютера

Физическая память компьютера представляет набор микросхем, припаяных к плате, в то время как внутренняя память компьютера состоит из регистров. Каждый регистр в памяти представляет собой одно место хранения, которое также называется ячейкой памяти. Ячейка памяти идентифицируется с использованием адреса. Ячейки и адреса памяти определяют, как организована память компьютера, чтобы пользователь мог эффективно хранить или извлекать информацию из компьютера.

Физическая память компьютера

При работе с микропроцессорами 80х86 архитектуры, различают три типа адресов: логический, линейный и физический.

Память, которую процессор адресует на своей шине, называется физической памятью. Каждому байту назначается уникальный адрес, называемый физическом адресом. Этот адрес используется для адресации ячеек памяти в микросхемах памяти. Физические адреса представлены в виде 32-битных или 36-битных целых чисел без знака. Их можно получить с помощью логического и линейного адреса.

Память, которую процессор адресует на своей шине, называется физической памятью. Каждому байту назначается уникальный адрес, называемый физическом адресом. Этот адрес используется для адресации ячеек памяти в микросхемах памяти. Физические адреса представлены в виде 32-битных или 36-битных целых чисел без знака. Их можно получить с помощью логического и линейного адреса.

Линейный адрес представляет собой одно 32-разрядное целое число без знака, которое можно использовать для адресации до 4 ГБ, то есть до 2^32 ячеек памяти. Линейные адреса обычно представляются в шестнадцатеричной системе счисления; их значения могут варьироваться от 0x00000000 до 0xffffffff. Линейный адрес можно получить с помощью преобразования логического адреса механизмом сегментации.

Благодаря адресации памяти у разработчика есть возможность контролировать память, затрачиваемую на выполнение действий программы и не давать ей превышать объем оперативной памяти устройства.
