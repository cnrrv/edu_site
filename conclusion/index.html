Выводы

    Память компьютера представляет из себя набор ячеек, каждая из которых идентифицируется в системе с помощью адреса - числового идентификатора, благодаря которому компьютер может обращаться к данным, размещенным в ячейках памяти.

    Методы адресации принято классифицировать на два больших класса: явную и неявную адресацию.

При использовании метода явной адресации в инструкции указывается адрес или адресный код операнда.

В методе неявной адресации не указывается адрес операнда - в инструкциях используется переменные или указатели.

В языке ассемблера при реализации программы есть возможность использовать удобный метод адресации. При использовании явной адресации в инструкции можно обращаться к ячейке памяти по её адресу при прямом методе адресации или с помощью смещения при косвенном методе адресации.

В языке программирования С при работе с памятью используется метод неявной адресации памяти. Во время работы со статической и динамической памятью применяется непосредственный метод адресации. При работе со стеком и его обработкой используется стековый метод адресации.

Для разработки импортонезависимого программного обеспечения и приложений требуются глубокие знания в архитектуре ЭВМ. Методы адресации памяти являются ее неотьемлемой частью, их знание и применение позволит разработчикам реализовывать качественные, быстродействующие и защищенные продукты.
